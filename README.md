# 🌦️ Weather App

A simple and responsive **Weather Application** built using **HTML, CSS, and JavaScript**. This app fetches real-time weather data and displays temperature, humidity, wind speed, and cloud conditions for any city.

---

## 🌐 Live Demo
https://weather-app-rho-nine-qy992zx05g.vercel.app/
---

## ✨ Features

- 🌍 Get weather by **current location**
- 🔍 Search weather by **city name**
- 🌡️ Displays temperature in Celsius
- 💨 Wind speed information
- 💧 Humidity percentage
- ☁️ Cloud conditions
- 🎨 Clean and minimal UI
- 📱 Responsive design

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Fetch API)

---
